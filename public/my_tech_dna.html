The Genesis of ENG-Portal

This space serves as a "Digital Conscious Mirror," mapping the technical evolution from a simple cloud UI to a robust, bare-metal Hybrid Edge-Cloud architecture.

Phase 1: Cloud-Only Era

Initial static web interface utilizing public Cloud LLMs. Heavy reliance on Tailwind and high-latency endpoints.

Phase 2: Local Edge Shift

Migration to a local Proxmox VE hypervisor (srv-1) with GPU passthrough (RTX 5060 Ti) for localized AI models.

Phase 3: The CORS Crisis

Encountered strict browser security blocks when the public layout attempted to fetch data from the local firewalled lab node.

Phase 4: Hybrid Router

Engineered the Secret Nexus Proxy (Node.js) to bypass CORS and intelligently route traffic between local Edge AI and Google Cloud.

Phase 5: Component Architecture

Complete refactoring to Vanilla Web Standards. Zero framework lock-in, semantic HTML5, and global CSS Design Tokens (Material Design 3).
